    CASOS DE USO
- [x] Você deseja criar pontos na história da produção do seu projeto;
- [x] Você deseja verificar mudanças feitas no seu projeto;

- [x] Você começa uma nova funcionalidade no seu projeto, sem estragar o que já foi feito;
- [x] Você adiciona as novas funcionalidade ao seu projeto em produção;
- [x] Você quer deletar a branch da nova funcionalidade, depois de aplicar em seu projeto;

- [x] Você quer colocar seu projeto na nunvem;

    COMANDOS
 - 'git init'   // Inicia a linha do tempo;
 - 'git add'    // Adiciona ou atualiza mudanças para irem para a linha do tempo;
 - 'git commit' // Adiciona um ponto na linha do tempo;
 - 'git log'    // Visualiza os pontos na linha do tempo;
 - 'git status' // Informa o estado das alterações do nosso projeto;
 - 'git show'   // Apresenta determinado ponto na história
 - 'git remote -v' // Apresenta os repositórios remotos;
 - 'git push -u origin master' // quando for a primeira vez, é preciso criar a branch master no repositório online.
 - 'git config credential.helper store' // Usado para não precisar colocar login e senha todas vez que for upar pra nuvem.
